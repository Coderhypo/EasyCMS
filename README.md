#EasyCMS

## 描述

一个极简的内容发布管理系统（CMS），使用Python的Flask框架，力求做一个通用性但不失轻量的CMS。

## 功能模块

### 用户

 + 添加用户
 + 用户管理

#### 权限分级

 + 管理员
 + 编辑
 + 作者
 + 订阅者

#### 权限划分

 + 修改网站信息（管理员）
 + 发布文章和页面（编辑及其以上）
 + 发布/编辑文章和页面（作者及其以上，作者修改会进草稿箱）
 + 添加分类（作者及其以上）
 + 分类管理（编辑及其以上）
 + 添加标签（作者及其以上）
 + 标签管理（编辑及其以上）
 + 查看隐私文章（订阅及其以上）

### 文章

 + 文章发布
 + 文章修改
 + 文章管理

### 页面
 
 + 页面发布
 + 页面修改
 + 页面管理

### 文章分类
 
 + 添加分类
 + 分类管理

### 标签

 + 添加标签
 + 标签管理
